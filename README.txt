Resolvendo captch usando redes neurais

Database obtida em: https://www.kaggle.com/fournierp/captcha-version-2-images

Instalar pacotes necessários usando:
conda create --name <env> -c conda-forge --file requirements.txt

Ativando o <env> criado:
conda activate <env>

No main.py selecionar a opção de treino ou predição e então executar:
python main.py
